Bienvenue sur planète babar !

Serveur de chat XMPP: babare.dynamic-dns.net. Le port par défaut est 5222, mais le serveur est également accessible au port 110 au cas où.

NOTE: le login termine toujours par @babare.dynamic-dns.net.

  • Gère ou crée ton compte xmpp
  • Admin serveur ejabberd
  • Client de chat (recommandé) :

    • Windows : gajim, pidgin (gaffe à l'installeur sourceforge..)
    • Android : Conversations
    • OSX : Adium
    • iPhone : lol

    • Client web Candy
